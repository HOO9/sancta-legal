계정 및 데이터 삭제 요청

성소(Sancta) 앱에서 계정 및 관련 데이터 삭제를 요청하는 방법을 안내합니다.

1. 앱 내 삭제 (권장)

  1. 성소 앱을 실행하고 로그인합니다
  2. 설정 → 계정 → 계정 삭제 버튼 탭
  3. 확인 절차 완료 → 즉시 삭제 처리

2. 이메일로 요청

앱 접근이 어려운 경우 아래 이메일로 요청하세요.

문의 이메일: heemin1010@gmail.com
제목: [성소] 계정 삭제 요청
본문: 가입 시 사용한 이메일 주소 기재

3. 삭제되는 데이터

  • 계정 정보 (이메일, 이름, 프로필 이미지)
  • 기도 기록 및 음성 녹음 파일
  • 수집한 말씀 카드 해금 기록
  • 대리 기도 요청 내역
  • 기도 광장에 작성한 게시물 및 댓글
  • 앱 사용 분석 데이터

4. 보관되는 데이터

  • 법령상 보관 의무가 있는 데이터 — 최대 5년
  • 익명화된 통계 데이터 — 개인 식별 불가능한 형태로 유지될 수 있음

5. 처리 기간

요청 접수 후 최대 30일 이내에 삭제 처리됩니다. 이메일 요청 시 확인을 위해 추가 본인 인증을 요청할 수 있습니다.

6. 삭제 후 영향

  • 계정 복구 불가능 — 동일 이메일로 재가입은 가능하나 과거 기록은 복원되지 않습니다
  • 진행 중인 대리 기도 요청은 자동으로 해제됩니다
  • 기도 광장 게시물은 삭제되며 다른 사용자에게도 더 이상 노출되지 않습니다
